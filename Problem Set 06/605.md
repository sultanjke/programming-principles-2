# Problem 605: 514505. Has a Vowel

Read one string *S*. Using any(), check whether *S* contains at least one English vowel (a, e, i, o, u) in any case. Print *Yes* or *No*.

## Input format
A single string *S* (without spaces).

## Output format
One line: Yes if there is at least one vowel, otherwise No.

## Examples

### Input
hello

### Output
Yes

### Input
rhythm

### Output
No
