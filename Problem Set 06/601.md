# Problem 601: 514501. Sum of Squares

Read an integer *n* and then *n* integers. Using map(), square each number and output the sum of all squares.

## Input format
First line: integer *n*. Second line: *n* integers.

## Output format
A single integer: the sum of squares of all given numbers.

## Examples

### Input
```
5
1 2 3 4 5
```

### Output
55

### Input
```
3
-1 0 2
```

### Output
5
