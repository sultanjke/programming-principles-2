# Problem 604: 514504. Dot Product

Read an integer *n*, then two arrays *A* and *B* of length *n*. Using zip(), compute and output the dot product $\sum_{i=1}^{n} i$ $A_i$ * $B_i$.

## Input format
First line: integer *n*. Second line: *n* integers of A. Third line: *n* integers of *B*.

## Output format
A single integer: the dot product of the two arrays.

## Examples

### Input
```
3
1 2 3
4 5 6
```

### Output
32

### Input
```
1
7
8
```

### Output
56
