# Problem 610: 514510. Count Truthy Numbers

Read an integer *n* and then *n* integers. Using map() with bool and sum(), count how many numbers are truthy (non-zero).

## Input format
First line: integer *n*. Second line: *n* integers.

## Output format
A single integer: the count of non-zero numbers.

## Examples

### Input
```
5
0 1 2 0 3
```

### Output
3

### Input
```
3
0 0 0
```

### Output
0
