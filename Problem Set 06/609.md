# Problem 609: 514509. Dictionary Query with Zip

Read an integer *n*, then *n* keys and *n* values. Build a dictionary using zip(keys, values). Then read one query key and output its value, or Not found if the key is absent.

## Input format
First line: integer *n*. Second line: *n* keys. Third line: *n* values. Fourth line: query key.

## Output format
The corresponding value, or Not found.

## Examples

### Input
```
3
a b c
1 2 3
b
```

### Output
2

### Input
```
2
x y
cat dog
z
```

### Output
Not found
