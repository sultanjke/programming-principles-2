# Problem 606: 514506. All Non-Negative

Read an integer *n* and then *n* integers. Using all(), check whether every number is non-negative. Print *yes* or *no*.

## Input format
First line: integer *n*. Second line: *n* integers.

## Output format
One line: Yes if all numbers are non-negative, otherwise No.

## Examples

### Input
```
5
0 1 2 3 4
```

### Output
Yes

### Input
```
3
-1 0 1
```

### Output
No
