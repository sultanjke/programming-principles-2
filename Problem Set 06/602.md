# Problem 602: 514502. Count Even Numbers

Read an integer *n* and then *n* integers. Using filter(), keep only even numbers and output how many there are.

## Input format
First line: integer *n*. Second line: *n* integers.

## Output format
A single integer: the number of even elements.

## Examples

### Input
```
5
1 2 3 4 5
```

### Output
2

### Input
```
4
2 4 6 8
```

### Output
4
