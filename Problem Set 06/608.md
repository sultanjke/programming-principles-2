# Problem 608: 514508. Sorted Unique Numbers

Read an integer *n* and then *n* integers. Using set() and sorted(), print all distinct numbers in increasing order, separated by spaces.

## Input format
First line: integer *n*. Second line: *n* integers.

## Output format
All distinct numbers in increasing order, separated by one space.

## Examples

### Input
```
5
3 1 2 3 2
```

### Output
1 2 3

### Input
```
4
10 10 10 10
```

### Output
10
