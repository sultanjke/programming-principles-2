# Problem 607: 514507. Longest Word

Read an integer *n* and then *n* words. Using max(..., key=len), output the longest word. If several words have the same maximum length, output the first one.

## Input format
First line: integer *n*. Second line: *n* words.

## Output format
A single word: the first longest word.

## Examples

### Input
```
4
cat tiger dog lion
```

### Output
tiger

### Input
```
3
a aa aaa
```

### Output
aaa
