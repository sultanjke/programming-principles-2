# Problem 603: 514503. Index Each Word

Read an integer *n* and then *n* words. Using enumerate(), output pairs in the format index:word, separated by one space.

## Input format
First line: integer *n*. Second line: *n* words.

## Output format
One line with *n* items: 0:w0 1:w1 ... (n-1):w(n-1)

## Examples

### Input
```
3
cat dog fish
```

### Output
0:cat 1:dog 2:fish

### Input
```
1
hello
```

### Output
0:hello
