# Problem 305: 506934. Shape and Square Area

Define a class Shape with a method area that returns 0. Define a subclass Square that inherits from Shape. The Square class should have a length attribute and override the area method to return *length^2.*

## Input format
An integer *n* representing the length of a square.

## Output format
The area of the square.

## Examples

### Input
0

### Output
0

### Input
5

### Output
25
