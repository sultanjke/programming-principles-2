# Problem 309: 506942. Circle Area

Define a Circle class that takes a radius. Implement an area method. Use *pi* = **3.14159.**

## Input format
An integer *r*.

## Output format
The area of the circle formatted to 2 decimal places.

## Examples

### Input
1

### Output
3.14

### Input
5

### Output
78.54

## Notes
Use print(f"{area:.2f}") to output exactly 2 decimal places.
More info here: https://docs.python.org/3/tutorial/inputoutput.html#formatted-string-literals
