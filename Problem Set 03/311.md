# Problem 311: 506945. Pair Addition

Define a Pair class with two integer values (*a* and *b*). Create a method add that takes another Pair object and returns their sum.

## Input format
Four integers *a1*, *b1*, *a2*, *b2*.

## Output format
The sum in the format: Result: [a_sum] [b_sum]

## Examples

### Input
1 2 3 4

### Output
Result: 4 6

### Input
0 0 1 1

### Output
Result: 1 1
