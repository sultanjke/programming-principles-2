# Problem 313: 506940. Filter Primes via Lambda

Given a list of numbers, use a lambda expression and the filter() function to create a new list containing only the prime numbers. If no primes are found, output No primes.

## Input format
A line of space-separated integers.

## Output format
A line of space-separated prime numbers, or No primes if none are found.

## Examples

### Input
1 2 3 4 5

### Output
2 3 5

### Input
10 11 12 13

### Output
11 13
