# Problem 304: 506932. String Upper-Case Class

Define a class named StringHandler which has two methods: getString to accept a string from console input using input() and printString to print the string in upper case.

## Input format
A single string *s*.

## Output format
The string *s* converted to all uppercase characters.

## Examples

### Input
hello world

### Output
HELLO WORLD

### Input
Python3

### Output
PYTHON3
