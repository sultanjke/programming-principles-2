# Problem 206: 421466. Maximum Number

Given a sequence consisting of *n* integers, write a program that outputs the maximum in this sequence.

## Input format
The first line contains an integer *n* (1 <= *n* <= 10^5) - the amount of integers. The next line contains *n* integers $a_i$ (-10^9 <= $a_i$ <= 10^9) - elements of the sequence.

## Output format
Single integer - the maximum in the sequence.

## Examples

### Input
4
2 7 3 3

### Output
7

### Input
1
3

### Output
3

### Input
3
-1 9 5

### Output
9

### Input
4
-10 -2 -5 -8

### Output
-2

### Input
2
1 8

### Output
8
