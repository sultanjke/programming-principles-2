# Problem 216: 195728. Newbie.

The element of the array is called a *newbie* if there was no same element in the array before. For every element of the array, print “YES” if it is a newbie, and “NO” otherwise.

## Input format
The first line of input contains an integer *n* (1 <= *n* <= 10^4). The next line contains *n* integers — elements of the array *a* (1 <= $a_i$ <= 10^6).

## Output format
For every element of the array, print “YES” if it is a newbie, and “NO” otherwise.

## Examples

### Input
6
1 4 10 4 1 9

### Output
YES
YES
YES
NO
NO
YES

### Input
3
1 2 3

### Output
YES
YES
YES

### Input
5
1 1 1 1 1

### Output
YES
NO
NO
NO
NO

### Input
7
100 500 42 100 21 100500 6005

### Output
YES

YES
