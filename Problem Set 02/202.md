# Problem 202: 72778. Sum

In this task, you should find the sum of all numbers from 1 to *n*.

## Input format
Given the positive integer *n*. (1 <= *n* <= 1000)

## Output format
Output the sum.

## Examples

### Input
3

### Output
6

### Input
10

### Output
55

### Input
1

### Output
1

### Input
7

### Output
28

### Input
100

### Output
5050
