# Problem 107: 503032. The Remainder

Read two integers **a** and **b**. Calculaet and print the remainder of the division of **a** by **b**.

## Input format
Two integers **a** and **b**.

## Output format
An integer representing the remainder of the division of **a** by **b**.

## Examples

### Input
2
3

### Output
8

### Input
5
0

### Output
1
