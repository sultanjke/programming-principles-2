# Problem 417: 510718. Radar Coverage

A radar is placed at point (0, 0) with coverage radius *R*. An airplane flies along a straight segment from point A($x_1$, $y_1$) to point B($x_1$, $y_1$). Find the length of the trajectory segment that lies inside (or on the boundary of) the radar circle.

## Input format
The first line contains real number *R*. The second line contains $x_1$, $y_1$. The third line contains $x_2$, $y_2$

## Output format
Print one real number: the required length. Absolute or relative error of 10^-6 is accepted.

## Examples

### Input
```
5
-10 0
10 0
```

### Output
10.0000000000

### Input
```
3
4 0
8 0
```

### Output
0.0000000000

### Input
```
4
0 0
3 0
```

### Output
3.0000000000
