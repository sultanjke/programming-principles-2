# Problem 413: 510714. Query Engine

You are given a JSON value and several queries. Each query is a path expression like user.friends[2].name or orders[0].items[1].price. Resolve each query and print the found value. Dot accesses object keys, and brackets access array indices.

## Input format
The first line contains a JSON value **J**. The first line contains a JSON value *q* - the number of queries. Each of the next *q* lines contains one query.

## Output format
For each query print one line. If the query is valid, print the resulting value serialized as compact JSON. If the query cannot be resolved, print NOT_FOUND.

## Examples

### Input
```
{"user":{"friends":[{"name":"Ann"},{"name":"Bob"},{"name":"Eve"}]}}
3
user.friends[0].name
user.friends[2].name
user.friends[5].name
```

### Output
```
"Ann"
"Eve"
NOT_FOUND
```

### Input
```
{"orders":[{"items":[{"price":10},{"price":20}]}]}
2
orders[0].items[1].price
orders[0].items[0].price
```

### Output
```
20
10
```

### Input
```
{"a":{"b":{"c":1}},"x":[10,20]}
4
a.b.c
x[1]
x[2]
a.b
```

### Output
```
1
20
NOT_FOUND
{"c":1}
```
