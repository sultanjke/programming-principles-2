# Problem 401: 510698. Squares of Numbers

Create a generator that yields squares of numbers from *1* up to *N*.

## Input format
A single integer *N*.

## Output format
Squares of all numbers from *1* to *N*, each on a new line.

## Examples

### Input
5

### Output
```
1
4
9
16
25
```

### Input
1

### Output
1

### Input
10

### Output
```
1
4
9
16
25
36
49
64
81
100
```
