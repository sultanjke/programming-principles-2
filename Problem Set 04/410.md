# Problem 410: 510711. Limited Cycle

Write a generator function that takes a list and an integer *k*, and cycles through and prints the list elements exactly *k* times.

## Input format
The first line contains a list of elements separated by space. The second line contains an integer *k*.

## Output format
The elements of the list repeated *k* times, all on one line separated by spaces.

## Examples

### Input
1 2 3
2

### Output
1 2 3 1 2 3

### Input
A B
3

### Output
A B A B A B

### Input
1
5

### Output
1 1 1 1 1
