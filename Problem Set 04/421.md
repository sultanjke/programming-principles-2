# Problem 421: 510722. Module Query Engine

You are given queries with a module path and an attribute name. For each query, import the module (module paths may be dotted, such as json.encoder or collections.abc) and classify the attribute. Output MODULE_NOT_FOUND if module import fails, ATTRIBUTE_NOT_FOUND if the module has no such attribute, CALLABLE if attribute exists and is callable, otherwise VALUE.

## Input format
The first line contains integer *q* - number of queries. Each of the next *q* lines contains module_path attribute.

## Output format
For each query print one of: MODULE_NOT_FOUND, ATTRIBUTE_NOT_FOUND, CALLABLE, VALUE.

## Examples

### Input
```
3
math sqrt
math pi
math missing_attr
```

### Output
```
CALLABLE
VALUE
ATTRIBUTE_NOT_FOUND
```

### Input
```
4
json dumps
json encoder
json.decoder JSONDecoder
json.decoder missing
```

### Output
```
CALLABLE
VALUE
CALLABLE
ATTRIBUTE_NOT_FOUND
```

### Input
```
5
collections.abc Iterable
collections.abc Mapping
collections.abc fake
datetime datetime
datetime timezone
```

### Output
```
CALLABLE
CALLABLE
ATTRIBUTE_NOT_FOUND
CALLABLE
CALLABLE
```
