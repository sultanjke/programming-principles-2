# Problem 412: 510713. Deep Diff

You are given two JSON objects. Find all differences between them recursively by object fields. For each different field print one line in the format path : old_value -> new_value, where path is dot-separated (for example, user.address.city). If a key exists only in one object, use <missing> for the absent side. Serialize values as compact JSON literals. Print lines sorted lexicographically by path. If there are no differences, print No differences.

## Input format
The first line contains JSON object **A**. The second line contains JSON object **B**.

## Output format
Print all differences as described, one per line, sorted by path. If there are no differences, print No differences.

## Examples

### Input
```
{"a":1,"b":2}
{"b":null,"c":3}
```

### Output
{"a":1,"c":3}

### Input
```
{"user":{"name":"Ann","age":20},"active":true}
{"user":{"age":21},"active":false}
```

### Output
{"active":false,"user":{"age":21,"name":"Ann"}}

### Input
```
{"x":{"y":{"z":1}},"k":5}
{"x":{"y":{"z":null,"w":2}},"k":{"m":1}}
```

### Output
{"k":{"m":1},"x":{"y":{"w":2}}}
