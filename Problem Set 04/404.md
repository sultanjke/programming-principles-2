# Problem 404: 510706. Squares from A to B

Implement a generator called squares(a, b) that yields the square of all numbers from *a* to *b*. Test it with a **for** loop and print each of the yielded values.

## Input format
Two integers *a* and *b*.

## Output format
The squares of numbers from *a* to *b* inclusive, each on a new line.

## Examples

### Input
1 5

### Output
```
1
4
9
16
25
```

### Input
5 10

### Output
```
25
36
49
64
81
100
```
