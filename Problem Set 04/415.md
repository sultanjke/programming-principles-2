# Problem 415: 510716. Next Birthday (Time Zones)

You are given a birth date and a current date, both in format YYYY-MM-DD UTC\pmHH:MM. Treat both as local midnight 00:00:00 in their respective time zones. Consider birthday by month and day. Find the nearest birthday moment that is not earlier than the current moment, and print how many days are left. The answer is **|∆| / 86400**, where ∆ is the difference in seconds. If ∆ = 0, print 0. If the birth date is February 29 and a target year is not leap, use February 28 in that year.

## Input format
The first line contains birth date with time zone. The second line contains current date with time zone. Both lines have format YYYY-MM-DD UTC\pmHH:MM.

## Output format
Print one integer: the number of days left until the nearest upcoming birthday moment.

## Examples

### Input
```
2000-05-10 UTC+00:00
2026-05-10 UTC+00:00
```

### Output
0

### Input
```
2000-05-10 UTC+00:00
2026-05-09 UTC+00:00
```

### Output
1

### Input
```
2000-05-10 UTC+00:00
2026-05-11 UTC+00:00
```

### Output
364
