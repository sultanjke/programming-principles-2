# Problem 414: 510715. Days Between (Time Zones)

You are given two date-time moments in the format YYYY-MM-DD UTC\pmHH:MM. Each date is interpreted as local midnight 00:00:00 in its own time zone. Compute the absolute difference between the two moments in full days.

## Input format
The first line contains one date-time moment. The second line contains another date-time moment. Each line has format YYYY-MM-DD UTC\pmHH:MM.

## Output format
Print one integer: the absolute difference in full days, computed as **|∆| / 86400**.

## Examples

### Input
```
2025-01-01 UTC+00:00
2025-01-02 UTC+00:00
```

### Output
1

### Input
```
2025-01-01 UTC+03:00
2025-01-01 UTC+00:00
```

### Output
0

### Input
```
2024-12-31 UTC-05:00
2025-01-01 UTC+02:00
```

### Output
0
