# Problem 420: 510721. Scope Accumulator

Process commands using Python variable scopes. There is a global variable g, initially 0. Inside function outer, there is variable n, initially *0*. Inside nested function inner, apply commands: global x adds x to g; nonlocal x adds x to n; local x changes only an inner local variable and does not affect g or n. After all commands, output final values of g and n.

## Input format
The first line contains integer *m* — number of commands. Each of the next *m* lines contains a command in format scope value, where scope is one of global, nonlocal, local.

## Output format
Print two integers: final g and final n.

## Examples

### Input
```
3
global 5
nonlocal 2
local 10
```

### Output
5 2

### Input
```
5
nonlocal 1
nonlocal 2
global 3
local 4
global -1
```

### Output
2 3

### Input
```
4
local 7
local -3
local 100
local 0
```

### Output
0 0
