# Problem 406: 510707. Fibonacci Generator

The Fibonacci Sequence is computed based on the following formula: f(n) = 0 if n = 0; f(n) = 1 if n = 1; f(n) = f(n-1) + f(n-2) if n > 1. Please write a program using a generator to print the Fibonacci sequence in comma separated form with a given *n* input by console.

## Input format
An integer *n* representing the count of elements.

## Output format
The first *n* Fibonacci numbers separated by commas.

## Examples

### Input
7

### Output
0,1,1,2,3,5,8

### Input
0

### Output
''

### Input
1

### Output
0
