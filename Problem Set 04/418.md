# Problem 418: 510719. Mirror Reflection

A light ray starts at point *A($x_1$, $y_1$)*, reflects from the axis Ox (mirror), and then reaches point *B($x_1$, $y_2$)*. Find the reflection point on Ox that satisfies the law of reflection (angle of incidence equals angle of reflection).

## Input format
The first line contains $x_1$, $y_1$, of point *A*. The second line contains $x_2$, $y_2$ of point *B*.

## Output format
Print two real numbers *x, y* — the reflection point coordinates. For this problem, *y = 0*. Absolute or relative error of 10^-6 is accepted.

## Examples

### Input
```
0 2
4 2
```

### Output
2.0000000000 0.0000000000

### Input
```
-1 3
5 1
```

### Output
3.5000000000 0.0000000000

### Input
```
2 5
-2 1
```

### Output
-1.3333333333 0.0000000000
