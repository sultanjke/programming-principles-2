# Problem 403: 510705. Divisibility Check

Define a function with a generator which can iterate the numbers, which are divisible by both *3* and *4*, between *0* and a given range *n*.

## Input format
An integer *n*.

## Output format
Numbers divisible by both *3* and *4* from 0 to *n* separated by spaces.

## Examples

### Input
50

### Output
0 12 24 36 48

### Input
12

### Output
0 12
