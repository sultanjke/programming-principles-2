# Problem 402: 510704. Even Numbers Generator

Write a program that takes *n* as input and prints all even numbers between *0* and *n* in comma-separated form using a generator.

## Input format
A single integer *n*.

## Output format
Even numbers from 0 to *n* separated by commas.

## Examples

### Input
10

### Output
0,2,4,6,8,10

### Input
1

### Output
0

### Input
0

### Output
0
