# Problem 409: 510710. Powers of Two

Create a generator that yields powers of 2(2^0, 2^1, 2^2 and so on) up to 2^n.

## Input format
An integer *n*.

## Output format
Powers of 2 from 2^0 up to 2^n inclusive, separated by spaces.

## Examples

### Input
5

### Output
1 2 4 8 16 32

### Input
0

### Output
1

### Input
10

### Output
1 2 4 8 16 32 64 128 256 512 1024
