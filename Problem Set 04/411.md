# Problem 411: 510712. JSON Patch Update

You are given two JSON objects: a source object and a patch object. Apply the patch using these rules: if a key from patch is missing in source, add it; if a patch value is *null*, remove that key from source; if both values are JSON objects, apply the same rules recursively; otherwise, replace the source value. Print the resulting JSON object.

## Input format
The first line contains a JSON object *source*. The second line contains a JSON object *patch*.

## Output format
Print one JSON object: the patched result, serialized in compact JSON form with keys sorted lexicographically.

## Examples

### Input
1 2 3
2

### Output
1 2 3 1 2 3

### Input
A B
3

### Output
A B A B A B

### Input
1
5

### Output
1 1 1 1 1
