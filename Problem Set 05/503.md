# Problem 503: 512510. Count Pattern Matches

Read two lines: a string *S* and a pattern *P* (literal string, no special regex symbols). Using re.findall() or re.finditer(), count how many times *P* appears in *S* as a non-overlapping match. Output the count.

## Input format
First line: string *S*. Second line: pattern *P* (literal).

## Output format
A single integer: the number of non-overlapping occurrences of *P* in *S*.

## Examples

### Input
```
abababa
ab
```

### Output
3

### Input
```
aaaa
aa
```

### Output
2
