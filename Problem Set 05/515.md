# Problem 515: 512523. Double Every Digit

Read a string from the input. Using re.sub() with a replacement function (callable), replace each digit by that digit repeated twice. For example, 1 becomes 11, 9 becomes 99. Output the resulting string.

## Input format
A single line of text.

## Output format
The string with every digit doubled.

## Examples

### Input
123

### Output
112233

### Input
0

### Output
00
