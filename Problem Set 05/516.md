# Problem 516: 512524. Extract Name and Age

Read a single line in the format Name: X, Age: Y, where X and Y are non-empty strings (name and number). Using capturing groups in a regex, extract X and Y. Output them on one line: X Y.

## Input format
One line: Name: X, Age: Y.

## Output format
One line: X Y (name and age separated by space).

## Examples

### Input
Name: Alice, Age: 25

### Output
Alice 25

### Input
Name: Bob, Age: 30

### Output
Bob 30
