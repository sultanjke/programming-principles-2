# Problem 505: 512512. Starts With Letter, Ends With Digit

Read a string from the input. Using regex metacharacters ^ (start) and $ (end) along with character classes, check if the string starts with a letter (A–Z or a–z) and ends with a digit (0–9). Print Yes or No.

## Input format
A single line of text.

## Output format
One line: Yes or No.

## Examples

### Input
Hello1

### Output
Yes

### Input
abc9

### Output
Yes
