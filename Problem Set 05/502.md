# Problem 502: 512509. Contains Substring

Read two lines: first a string *S*, then substring *P*. Using re.search(), determine if *P* occurs in *S* (as a substring). Print *Yes* if it does, otherwise *No*.

## Input format
First line: string *S*. Second line: substring *P*.

## Output format
One line: Yes or No.

## Examples

### Input
```
hello world
world
```

### Output
Yes

### Input
```
python
py
```

### Output
Yes
