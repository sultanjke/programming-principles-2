# Problem 519: 512527. Word Count With Compiled Pattern

Read a string from the input. Using re.compile() to compile a pattern that matches a word (sequence of word characters with word boundaries), count how many words the string contains. Output the count.

## Input format
A single line of text.

## Output format
A single integer: the number of words.

## Examples

### Input
hello world

### Output
2

### Input
one

### Output
1
