# Problem 504: 512511. Extract All Digits

Read a string from the input. Using the special sequence d in a regular expression, find all digits in the string. Output them in order, separated by spaces. If there are no digits, output an empty line.

## Input format
A single line of text.

## Output format
All digits found, in order, space-separated (or an empty line if none).

## Examples

### Input
a1b2c3

### Output
1 2 3

### Input
no digits here

### Output
''
