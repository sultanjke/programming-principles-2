# Problem 506: 512513. First Email-Like Substring

Read a line of text. Using re.search() with a simple email-like pattern (non-whitespace, then @, then non-whitespace, then ., then non-whitespace), find the first such occurrence. Output the matched substring, or No email if none.

## Input format
A single line of text.

## Output format
The first email-like substring, or No email.

## Examples

### Input
Contact john.doe@example.com for info

### Output
john.doe@example.com

### Input
No at sign here

### Output
No email
