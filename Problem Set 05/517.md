# Problem 517: 512525. Count Date-Like Patterns

Read a string from the input. Using regular expressions, count how many date-like substrings of the form DD/MM/YYYY appear in it, where DD and MM are two digits and YYYY is four digits (format check only).

## Input format
A single line of text.

## Output format
A single integer: the number of DD/MM/YYYY matches.

## Examples

### Input
Today is 17/02/2025 and 18/02/2025

### Output
2

### Input
No dates

### Output
0
