# Problem 518: 512526. Literal Pattern Count

Read two lines: a string *S* and a pattern *P* that may contain regex metacharacters (e.g. ., *). Using re.escape() to treat *P* as a literal, count how many times *P* appears in *S*. Output the count.

## Input format
First line: string *S*. Second line: pattern *P* (to be used as literal).

## Output format
A single integer: the number of literal occurrences of *P* in *S*.

## Examples

### Input
```
a.b.c.d
.
```

### Output
3

### Input
```
***
*
```

### Output
3
