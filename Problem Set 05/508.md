# Problem 508: 512515. Split by Pattern

Read two lines: a string *S* and a regex delimiter pattern *D*. Using re.split(), split *S* by pattern *D* and output the resulting parts separated by commas. The pattern is a regular expression (not a literal delimiter).

## Input format
First line: string *S*. Second line: regex delimiter pattern *D*.

## Output format
Parts of *S* after splitting by regex pattern *D*, separated by commas.

## Examples

### Input
```
a,b;c|d
[,;|]+
```

### Output
```
a,b,c,d
```

### Input
```
one two three
\s+
```

### Output
```
one,two,three
```
