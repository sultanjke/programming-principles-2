# Problem 514: 512522. Digits Only With Compile

Read a string from the input. Using re.compile() to compile a pattern for a string consisting only of digits from start to end, check whether the whole string matches. Print Match if it does; otherwise print No match.

## Input format
A single line of text.

## Output format
One line: Match or No match.

## Examples

### Input
12345

### Output
Match

### Input
0

### Output
Match
