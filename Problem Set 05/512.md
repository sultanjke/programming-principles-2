# Problem 512: 512520. Sequences of Two or More Digits

Read a string from the input. Using the appropriate quantifier, find all contiguous sequences of at least 2 digits. Output them in order, separated by spaces. If there are none, output an empty line.

## Input format
A single line of text.

## Output format
All sequences of 2 or more digits in order, separated by spaces (or an empty line if none).

## Examples

### Input
a12b34c5

### Output
12 34

### Input
99 and 100

### Output
99 100
