# Problem 509: 512516. Count Three-Letter Words

Read a string from the input. Using regular expressions, count how many words in the string have length exactly 3, and output that count.

## Input format
A single line of text (words separated by spaces).

## Output format
A single integer: the number of words of length exactly 3.

## Examples

### Input
the cat and dog ran

### Output
5

### Input
one two three four

### Output
2
