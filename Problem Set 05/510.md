# Problem 510: 512517. Cat or Dog

Read a string from the input. Using the alternation metacharacter | in a regex, check whether the string contains the substring cat or the substring dog. Print Yes if it does, otherwise No.

## Input format
A single line of text.

## Output format
One line: Yes or No.

## Examples

### Input
I have a cat

### Output
Yes

### Input
I have a dog

### Output
Yes
