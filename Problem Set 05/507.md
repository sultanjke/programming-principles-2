# Problem 507: 512514. Replace All Occurrences

Read three lines: a string *S*, a pattern *P* (literal substring), and a replacement string *R*. Using re.sub(), replace every occurrence of *P* in *S* with *R*, and print the resulting string.

## Input format
First line: string *S*. Second line: pattern *P*. Third line: replacement *R*.

## Output format
The string *S* with all occurrences of *P* replaced by *R*.

## Examples

### Input
```
hello world
world
Python
```

### Output
```
hello Python
```

### Input
```
aaa
a
b
```

### Output
```
bbb
```
