# Problem 513: 512521. Word Count With Regex

Read a string from the input. Using the special sequence w+ to match "words" (sequences of word characters), find all such matches and output their count.

## Input format
A single line of text.

## Output format
A single integer: the number of words (matches of w+).

## Examples

### Input
hello world

### Output
2

### Input
one

### Output
1
